# Alex-Discord-Moderation-Bot-
Author
Himanshu Sharma

DevDungeon is a powerful and feature-rich Discord moderation bot designed to help server administrators maintain order, manage users, and retrieve useful server and user information with ease.
Built with scalability and performance in mind, this bot provides essential moderation, utility, and server management commands.

🚀 Features
Moderation Commands

- warn – Warn a user
- warnings – View a user’s warnings
- clearwarnings – Clear all warnings of a user
- mute – Mute a user
- unmute – Unmute a user
- kick – Kick a user from the server
- ban – Ban a user
- unban – Unban a user
- purge – Delete multiple messages at once

ℹ️ User Information

- avatar – Get a user’s avatar
- banner – Get a user’s banner
- userinfo – View detailed information about a user

🏠 Server Information

- servericon – View the server icon
- serverbanner – View the server banner
- emojicount – Get total emoji count
- membercount – View server member count
- serverinfo – Get detailed server information

🧰 Utility Commands

- ping – Check bot latency
- setlogchannel – Set a moderation log channel
- help – View available commands and usage

⚙️ Requirements

- A Discord Bot Token
- Administrator or required moderation permissions
- Discord Privileged Gateway Intents enabled:
- Server Members Intent
- Message Content Intent


🔑 Permissions
- The bot is intended to run with Administrator permissions for full moderation capabilities.
  Ensure the bot’s role is above other roles it needs to manage.


📌 Use Cases

- Community moderation
- Spam control
- User management
- Server analytics & info
- Private or public Discord servers
